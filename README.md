# Teerex Store

- [Click here](https://dashing-churros-b0614a.netlify.app/) to see preview of my project.

## Tech Stack

- CSS 3
- Javascript
- React JS
- Context API + useReducer
- Docker
- Kubernetes
- Jenkins
- Webhooks

## Features

- Product Page
- Filter
- Add to cart

## Run Locally

Clone the project

```bash
  git clone https://github.com/Manoranjan-Sethi/teerex-store.git
```

Go to folder

```bash
 cd teerex-store
```

Install packages

```bash
 npm i
```

Run the app

```bash
  npm start
```
